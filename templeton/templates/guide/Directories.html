1.4 Directories ¶

~/pads
bin/

The PADS Binary File

src/msg.sh

The functions for the error messages, argRequired, and argIllegal

src/P src/S src/R

Directory used for the each respective module. To make things easy to code, the directory is named the same as the flag in the CLI.

/usr/local/man/man1/pads.1.gz

Where the pads.1.gz file lives. For the man page

/usr/share/info/pads.info.gz

Where the pads.info.gz file lives. For the info page

/srv/http

For websites in production

$HOME/.config/pads/

For personal files that PADS needs to use
