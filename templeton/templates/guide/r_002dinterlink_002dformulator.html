2.5.4 interlink.formulator ¶

A collection of tools used to generate Forms, Reports, and Navigation Bars from your database

  • interlink.formulator.Generator(db, user, password)

2.5.4.1 interlink.formulator.Generator(db, user, password) ¶

Create the object for generating you form, report, query, or nav. 

class Generator: 
  def __init__(self, db, user, password):
    self.db = db
    self.user = user
    self.password = password
    
  def generateForm(self, table):
    error, form = createForm(self.db, self.user, self.password, table)
    title = table
    return error, form, title
    
  def generateReport(self, table):
    colName, colRow, error = createReport(self.db, self.user, self.password, table)
    title = table
    return colName, colRow, error, title
    
  def generateQuery(self, table):
    return createQuery(self.db, self.user, self.password, table)
    
  def generateNav(self):
    return createNav(self.db, self.user, self.password)
__init__(self, db, user, password)

Initalize the connection, create the object.

db

The name of the Environment Variable for the database

user

The name of the Environment Variable for the user which has access to the database

password

The name of the Environment Variable for the password of the user which has access to the database

generateForm(self, table)

Generate the Form, URL, and Template for said table in database. Variables returned are error form title

generateReport(self, table)

Generate the Report, URL, and Template for said table in database. Variables returned are error table colName colRow

generateQuery(self, table)

Generate a Query for said table in database. Unlike creating a URL and Template, you can create a variable to be used in a template.

generateNav(self, table)

Generate a Navigation Bar for said table in database. It basically just finds the table names, rather than the table data.
